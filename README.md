 ## Installation
 
```bash
composer install
```

## Key Generate

```bash
php artisan key:generate
```

## Data Base

```bash
php artisan migrate
```

copy .env.example to .env 
for windows 

```bash
copy .env.example .env
```
for linux 

```bash
cp .env.example .env
```
Now all set and run 

```bash
php artisan serv
```
